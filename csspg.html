CSS Cheat Sheet!

The table on the side shows a few CSS elements, properties, functionalities.

Font Properties Color and Background Properties Text Properties Box Properties
Font-Family - Changes the font family of certain words, sentences, paragraphs, etc. Color - Changes the color of text. Text-Decoration - Allows text to be decorated through one of five properties: underline, overline, line-through, blink, none. Margin - Sets the margins of an element by specifying top, bottom, left and right margins, all either specifying length or percentage.
Font-Style - Changes text: normal, oblique, and italics. Background-Image - Sets the background image of an element. Text-Transform - Allows for capitalizing the first letter of each word (capitalize), capitalizing all letters of a word (uppercase), using all small letters in each word(lowercase), and the inital value(none). Padding - Shorthand for the padding-top, padding-right, padding-bottom, and padding-left properties.
Font-Weight - Used to specify the weight of the font. Background-Attachment - Determines if a specified background image will scroll with the content or be fixed with regard to the canvas. Width - Each block-level or replaced element can be given a width, specified as a length, a percentage, or as auto.
