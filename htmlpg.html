HTML Cheat Sheet!

The table on the side shows a few HTML elements, properties, functionalities.

Basic Tags Formatting Lists Links
The "html" tag - Creates an HTML document. The "p" tag - Creates a new paragraph. The "ul" tag - Creates an unordered list. The "a href='URL'" tag - Creates a hyperlink to a Uniform Resource Locator
The "head" tag - Sets off the title & other info that isn't displayed. The "br" tag - Inserts a line break (carriage return) The "ol" or "ol start=?" tag - Creates an ordered list (start=xx, where xx is a counting number) The "a href='mailto:EMAIL_ADDRESS'" tag - Creates a hyperlink to an email address
The "body" tag - Sets off the visible portion of the document. The "blockquote" tag - Puts content in a quote - indents text from both sides The "li" tag - Encompasses each list item The "a name='NAME'" tag - Creates a target location within a document
The "title" tag - Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked. The "div" tag - Used to format block content with CSS The "dl" tag - Creates a definition list The "a href='#NAME'" tag - Creates a link to that target location
